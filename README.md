jquery-flashbox
===============

Flashes the background color, and / or border color, and / or text color of an element

##Usage
```javascript
$("#selector").flashBox({
  backgroundColor: 'rgb(249, 206, 0)',
  borderColor: 'rgb(222, 184, 0)',
  color: 'rgb(256, 256, 256)',
  flashes: 5,
  totalDuration: 4000
});
```
